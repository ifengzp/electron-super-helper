# Electron-super-helper

一款基于electron写的小工具箱，用来存放一些自己常用的工具集合；

在平时有些挺有用的办公工具比如截图取字，可以大大提高效率，就顺手写一个将就用一下，目前功能只有两个：
- 截图取字
- PDF转图

目前除了这两个功能能够大大提升我的效率外，还没想到要添加什么其他功能，后面如果再遇到痒点会慢慢往上面造轮子，大于五个会考虑重构下代码
