webfont 字体预览

{{ text }}

第一步：使用font-face声明字体 
          @font-face {
              font-family: {{ fontFamily }};
              font-display: swap;
              src: url('{{ fontFamily }}.eot'); /* IE9 */
              src: url('{{ fontFamily }}.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
              url('{{ fontFamily }}.woff2') format('woff2'),
              url('{{ fontFamily }}.woff') format('woff'), /* chrome、firefox */
              url('{{ fontFamily }}.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
              url('{{ fontFamily }}.svg#webfont') format('svg'); /* iOS 4.1- */
          }
第二步：定义使用 webfont 的样式 
          .web-font {
              font-family: {{ fontFamily }} !important;
              font-size: 16px;
              font-style: normal;
              -webkit-font-smoothing: antialiased;
              -moz-osx-font-smoothing: grayscale;
          }
第三步：为文字加上对应的样式 
<i class="web-font">{{ text }}</i>
